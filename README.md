## Install
```javascript
npm i @gaojuqian/object-test
```

## Use
```javascript
import { expect } from "@gaojuqian/object-test";


expect({ abc: 123 }).toEqual({ abc: 123 });
// -> true

expect({ abc: 123 }).not.toEqual({ abc: 123 });
// -> false
```
